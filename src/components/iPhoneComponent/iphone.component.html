iPhone Download

Please follow the instructions below to download the T-Party Application for iPhone.

  1. Select the install link below.

    Install T-Party for iPhone

  2. If you receive an Untrusted Enterprise Developer warning. Navigate to Settings > General > Profiles & Device Management > Slalom Consulting and select Trust

  3. When the alert view appears. Select Install

  4. Select the home button and locate T-Party App icon on your home screen

  5. Wait for install to complete and select the application to launch.
